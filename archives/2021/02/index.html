[Django] values()와 values_list()

Feb 25, 2021 in Django

values(), values_list() 둘 다 모델 객체에서 필요한 필드의 값을 가져올 때 사용하는 메소드지만,
리턴하는 쿼리셋의 결과에 차이점이 있으니 상황에 맞게 사용하자.

1. values()

  • 딕셔너리를 가진 쿼리셋을 반환한다. : {'필드명':'값', ...}
  • 필드를 지정하지 않으면 모든 필드를 반환하고, 필드를 지정하면 지정한 필드만 반환한다.

2. values_list()

  • 튜플을 가진 쿼리셋을 반환한다. : ('값', ...)
  • 필드를 지정하지 않으면 모든 필드를 반환하고, 필드를 지정하면 지정한 필드만 반환한다.

values_list()의 flat=True

  • 필드 외에 flat 파라미터를 True로 지정하면, 튜플이 아닌 단일값을 반환한다. : '값'
  • 이 경우에는 1개의 필드만 지정할 수 있다.

Reference : https://docs.djangoproject.com/en/3.1/ref/models/querysets/#values
